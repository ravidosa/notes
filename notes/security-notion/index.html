Security Notion¶

advantage of adversary between 0 and 1 (1 good, 0 bad) prove security notions equivalent/stronger through reduction if breaking A breaks B, A stronger than B reasonable (polytime) adversary hybrid argument one way to random point, other way past that point (triangle inequality) perfect privacy/LR security \(\mathrm{Pr}[C = C_0 | M - M_0] = \mathrm{Pr}[C = C_0 | M = M_1]\) \(L(\cdot, \cdot)\) encrypts left input, \(R(\cdot, \cdot)\) encrypt right input \(\mathrm{Adv}_{\pi}^{lr}(\mathcal{A}) = \mathrm{Pr}[\mathcal{A}^{L(\cdot,\cdot)} \to 1] - \mathrm{Pr}[\mathcal{A}^{R(\cdot,\cdot)} \to 1]\) perfect privacy if \(0\) advantage can't tell which input encrypted 

algorithm L(M0, M1): # left oracle
    if |M0| != |M1| then return None
    if undef(K) then K = K(l)
    C = E(K, M0)
    return C

algorithm R(M0, M1): # right oracle
    if |M0| != |M1| then return None
    if undef(K) then K = K(l)
    C = E(K, M1)
    return C
